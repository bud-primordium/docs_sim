Lennard-Jones 势计算的实现，包括截断区域的多项式处理

该文件包含用于 Lennard-Jones 势力和能量计算的函数，包括处理周期性边界条件和截断修正。

Author

Gilbert Young

Date

2024-10-20

Functions

void solve_cutoff_cubic(double cutoff, double r0, double sigma, double epsilon, double &a, double &b, double &c, double &d)

使用三次多项式计算截断区域的系数 a, b, c, d

满足以下条件：

  1. V(r0) = V0

  2. V’(r0) = F0

  3. V(cutoff) = 0

  4. V’(cutoff) = 0

参数:

  • cutoff – 截断半径

  • r0 – 过渡起点

  • sigma – LJ势参数 σ

  • epsilon – LJ势参数 ε

  • a – 三次项系数

  • b – 二次项系数

  • c – 一次项系数

  • d – 常数项

void calculate_lj_forces(int num_atoms, const double *positions, double *forces, double epsilon, double sigma, double cutoff, const double *box_lengths, const int *neighbor_pairs, int num_pairs)
double calculate_lj_energy(int num_atoms, const double *positions, double epsilon, double sigma, double cutoff, const double *box_lengths, const int *neighbor_pairs, int num_pairs)

计算 Lennard-Jones 势能，使用最小镜像法处理周期性边界条件

参数:

  • num_atoms – 原子数量

  • positions – 原子位置数组（长度为 3*num_atoms）

  • epsilon – Lennard-Jones 势参数 ε，单位为 eV

  • sigma – Lennard-Jones 势参数 σ，单位为 Å

  • cutoff – 截断半径，单位为 Å

  • box_lengths – 模拟盒子在每个维度的长度（长度为 3）

返回:

总 Lennard-Jones 势能，单位为 eV

Nose-Hoover 恒温器的实现

该文件包含 Nose-Hoover 恒温器的实现，主要用于分子动力学模拟中的温度控制。 它通过引入单个热浴变量，遵循拓展哈密顿量，更新系统的速度和热浴变量，从而实现温度调节。
