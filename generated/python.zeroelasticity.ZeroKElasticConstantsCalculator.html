num_stepsint, optional

优化器类型，支持 ‘GD’（梯度下降）、’BFGS’、’LBFGS’，默认为 ‘LBFGS’

优化器的参数，默认为 None。若提供，将覆盖默认参数。

save_pathstr, optional
