class python.integrators.MTKIntegrator(thermostat=None, barostat=None, n_chains: int = 3, n_yoshida: int = 4)[源代码]

Martyna-Tobias-Klein (MTK) 积分器实现

该积分器专门设计用于NPT系综，可以正确处理热浴和压浴的耦合。 使用多时间尺度积分方案，确保系统的哈密顿量守恒。

Parameters:
thermostatOptional[Thermostat]

恒温器对象

barostatOptional[Barostat]

恒压器对象

n_chainsint

Nose-Hoover链的长度，默认为3

n_yoshidaint

Yoshida积分器的阶数，默认为4
